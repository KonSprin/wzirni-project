# Network Traffic Analysis Documentation

Welcome to the documentation for the encrypted network traffic analysis project.

## Overview

This project enables comprehensive analysis of encrypted network traffic using PolarProxy for TLS decryption,
Wireshark for packet analysis, and a Python-based client-server setup that generates realistic HTTPS traffic.

## Key Features

- **TLS Traffic Decryption**: Use PolarProxy to decrypt HTTPS traffic for deep inspection
- **Comparative Analysis**: Compare encrypted vs decrypted traffic side-by-side
- **Realistic Traffic Generation**: Client simulates real user behavior with varied patterns
- **Web-based Analysis**: Wireshark accessible through browser interface
- **Docker-based**: Fully containerized for easy deployment and reproducibility
